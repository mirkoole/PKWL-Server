                        Public-Key-Web-Login (PKWL)
                        	(Wordpress-Plugin)

Install
-------

Upload the PKWL.zip file using the wordpress plugin backend and activate the plugin.

That's all!


Uninstall
---------

Use wordpress plugin backend to run the uninstall process.
All plugin files and any data stored by the plugin will be removed.
